# JobSkillShare (JSS) Website Audit & Redesign — Week 1

## Purpose
JobSkillShare (jobskillshare.org) is an IT skills training platform offering hands-on courses, live labs, and certification prep (IT Support, Cloud, Networking, Cybersecurity) for individuals and organizations.

This repo documents an independent audit of the JSS website — no backend/live-site access — with the goal of identifying UX issues and building a prioritized improvement list to guide Week 2 prototyping.

## Objective
Improve and extend JSS's existing website: audit live pages, build prototypes for improvements, and develop campaign support pages. All work is built independently and submitted for review.

## Folder Structure
```
/audit          → Daily audit notes and findings (Days 1-4)
/screenshots    → Screenshots supporting each finding
/reports        → Weekly reports (e.g. Week 1 Report.docx)
/prototypes     → Week 2+ prototype files (added later)
```

## Week 1 Summary
| Day | Focus | Output |
|---|---|---|
| Day 1 | Dev environment setup + initial site review (structure, nav, visual hierarchy) | Site map + first-pass issue list |
| Day 2 | Homepage UX audit (CTA, above-the-fold, nav, mobile, load speed) | Homepage audit report |
| Day 3 | Program/course catalog audit (content, descriptions, enrollment flow) | Catalog audit report |
| Day 4 | Membership/registration audit (sign-up friction, trust signals) | Membership audit + suggested changes |
| Day 5 | Compiled top-10 priority list + consolidated Week 1 report | `reports/JSS_Week1_Report.docx` |
